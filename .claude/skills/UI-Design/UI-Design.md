# UI-Design Skill

## Purpose
The UI-Design skill ensures proper presentation and user experience by validating MDX layout, sidebar configuration, image integration, and responsive design elements. It maintains visual consistency and usability across the book project.

## Capabilities
- MDX layout validation
- Sidebar structure verification
- Image and media validation
- Responsive design checking
- Accessibility compliance

## Validation Checks

### 1. MDX Layout Validation
- Verify proper MDX syntax and structure
- Check for valid React component usage
- Validate code block formatting
- Ensure proper content organization

### 2. Sidebar Structure
- Verify sidebar configuration accuracy
- Check for proper navigation hierarchy
- Validate all document references exist
- Ensure consistent organization

### 3. Image and Media Integration
- Verify image file references are valid
- Check image dimensions and quality
- Validate alt text and accessibility
- Ensure proper image placement

### 4. Responsive Design
- Check layout adaptability across screen sizes
- Verify mobile-friendly navigation
- Validate readable text sizes
- Ensure proper spacing and alignment

### 5. Accessibility Compliance
- Verify proper heading hierarchy
- Check for sufficient color contrast
- Validate keyboard navigation support
- Ensure screen reader compatibility

## Example Triggers
- Layout change: Validate new or modified layouts
- Content addition: Verify proper integration
- Design update: Re-validate affected areas
- Accessibility check: Ensure compliance

## Guardrails
- Prevents publication of inaccessible content
- Blocks layouts that don't meet design standards
- Ensures responsive design compliance
- Maintains consistent visual presentation
